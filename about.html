About Me

Hai! Saya Raihan Andi Saungnaga, seorang mahasiswa Universitas Lampung yang sangat antusias dalam dunia teknologi, khususnya dalam bidang Machine Learning.

Saya menyukai bagaimana data dapat digunakan untuk memprediksi, menganalisis, dan memberikan solusi terhadap berbagai masalah nyata. Dengan menggunakan algoritma machine learning dan kecerdasan buatan, saya percaya bahwa banyak hal besar bisa diwujudkan dari otomasi, prediksi perilaku pengguna, hingga inovasi dalam berbagai industri.

Saat ini saya terus mengasah kemampuan saya dalam pengolahan data, eksplorasi algoritma supervised maupun unsupervised learning, serta pembuatan model-model prediktif menggunakan Python dan berbagai library populer seperti Scikit-Learn, Pandas, dan TensorFlow.
